Acerca de mi

Soy una persona qué le encanta solucionar problemas, alguien muy resolutivo, con buena capacidad de trabajo en equipo. Project Fast Food App

Consta de un servidor realizado en Express, utilizando cómo base datos postgres. En el manejo de sesiones entre usuarios y acceso al servidor utilizo JSON Web Token para así una mayor seguridad en el servidor. A la hora de digitar información sensible se encriptarán los datos. Manejo de ORM para una comunicación más sencilla y eficáz entre la base de datos y el servidor
